BoyerMooreAlgorithm.cs, находящийся в этой папке - изначальная версия алгоритма, без счетчика итераций.
В проекте Benchmarking.csproj находится вариант алгоритма с счетчиком итераций, а также код для генерации данных, измерения производительности и построения графиков.
Benchmarking.csproj - модифицированная версия проекта из практики Ulearn 1 семестра.
Данные находятся в Benchmarking.csproj\bin\Debug\input.txt.
Результаты измерений находятся в Benchmarking.csproj\bin\Debug\BenchmarkData.txt.
Графики есть в презентации, также можно запустить проект.
